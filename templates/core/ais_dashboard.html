How to Import AIS JSON File

To import the AIS (Annual Information Statement) JSON file, please follow these steps:

  1. Log in to the Income Tax Portal.
  2. Enter your PAN/User ID and Password.
  3. After logging in, navigate to the AIS section.
  4. Click on Annual Information Statement (AIS).
  5. Click on the Download tab.
  6. Select Annual Information Statement (AIS) – JSON (for AIS Utility).
  7. Download the JSON file.
Note: The downloaded JSON file is password-protected. The password is your PAN number in lowercase combined with your Date of Birth in DDMMYYYY format (e.g., abcde1234f03021982).
